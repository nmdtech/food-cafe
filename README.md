# 🍒 Cherry Party Cafe

A classic old-school cafe website UI built with **pure HTML and CSS**.

This project is a practice design focused on understanding the fundamentals of web layout without using any frameworks, libraries, or preprocessors.

## ☕ About The Project

Cherry Party is a vintage-style cafe interface inspired by old-school website designs.

The goal of this project was to practice:

- Writing clean HTML structure
- Building layouts with pure CSS
- Creating custom grids manually
- Understanding spacing, positioning, and alignment without frameworks
- Designing a UI from scratch without Bootstrap, Tailwind, or CSS libraries

## 🛠️ Built With

- HTML5
- Pure CSS3

No frameworks.  
No Sass.  
No external UI libraries.

Just the basics. 🍒

## 📸 Preview

A simple vintage cafe experience with a handmade layout and classic web design feeling.

## 📚 Purpose

This repository is mainly for learning and practicing traditional frontend development.
